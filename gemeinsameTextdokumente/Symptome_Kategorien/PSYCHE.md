#  Symptom-Katalog: Psychisches Erleben & Verhalten 


| Bereich | Symptom (Nutzer-Wahrnehmung) | Detail-Beschreibung für die Triage-Logik | 🚨 Red Flag |
| :--- | :--- | :--- | :--- |
| **Energie & Antrieb** | **Anhaltende Erschöpfung** | Gefühl von extremer Müdigkeit, die auch durch Schlaf nicht gebessert wird. | Nein |
| | **Antriebslosigkeit** | Massive Überwindung nötig, um einfachste Dinge (Essen, Aufstehen) zu tun. | Nein |
| | **Innere Erstarrung** | Gefühl, emotional "leer" oder wie versteinert zu sein; keine Gefühlsregungen mehr. | Nein |
| **Stimmung** | **Niedergeschlagenheit** | Tiefes Gefühl von Traurigkeit oder Sinnlosigkeit über einen langen Zeitraum. | Nein |
| | **Interessenverlust** | Dinge, die früher wichtig waren, fühlen sich jetzt völlig gleichgültig an. | Nein |
| | **Starke Selbstzweifel** | Ständige Gedanken über eigene Fehler oder Minderwertigkeit. | Nein |
| **Angst & Körper** | **Plötzliches Herzrasen** | Anfallartige Episoden mit Todesangst, Schweißausbrüchen und Atemnot. | Nein |
| | **Innere Unruhe** | Ein Gefühl von ständiger Getriebenheit oder "unter Strom" stehen. | Nein |
| | **Gedankenkreisen** | Unfähigkeit, das Grübeln über Probleme oder Sorgen zu stoppen. | Nein |
| **Wahrnehmung** | **Konzentrationsmangel** | Unfähigkeit, einem Gespräch zu folgen oder eine Seite in einem Buch zu lesen. | Nein |
| | **Veränderte Wahrnehmung** | Gefühl, dass die Umwelt oder man selbst "fremd" oder "unwirklich" erscheint. | Nein |
| **Akute Krise** | **Selbstverletzungsgedanken** | Der wiederkehrende Drang, sich selbst körperlichen Schmerz oder Schaden zuzufügen. | **JA** |
| | **Suizidgedanken** | Konkrete Pläne oder der Wunsch, das eigene Leben zu beenden. | **JA** |
| | **Realitätsverlust** | Hören von Stimmen oder Sehen von Dingen, die andere nicht wahrnehmen. | **JA** |
| | **Akute Verwirrtheit** | Plötzliches Nicht-Wissen, wer man ist, welches Datum wir haben oder wo man sich befindet. | **JA** |
| | **Fremdgefährdung** | Der Impuls oder das Vorhaben, anderen Menschen körperlich zu schaden. | **JA** |
