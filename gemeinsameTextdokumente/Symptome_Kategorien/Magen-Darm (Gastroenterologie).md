| Bereich | Symptom |
| :--- | :--- |
| **Speiseröhre** | **Sodbrennen:** Brennendes Gefühl hinter dem Brustbein, oft nach dem Essen oder im Liegen. |
| | **Saures Aufstoßen:** Rückfluss von Magensaft oder Speiseresten bis in den Mundraum. |
| | **Schluckstörung:** Das Gefühl, dass Nahrung im Hals oder in der Brust „stecken bleibt“. |
| | **Kloßgefühl:** Ein ständiges Druck- oder Fremdkörpergefühl im Halsbereich. |
| | **Schmerz beim Schlucken:** Stechender Schmerz direkt beim Schluckvorgang. |
| **Magen** | **Magendruck:** Unangenehmer Druck oder Schmerz direkt unter dem Brustbein (Oberbauch). |
| | **Völlegefühl:** Gefühl, bereits nach sehr kleinen Portionen „pappsatt“ zu sein. |
| | **Nüchternschmerz:** Magenschmerzen bei leerem Magen, die nach dem Essen oft besser werden. |
| | **Übelkeit & Erbrechen:** Flaues Gefühl im Magen bis hin zum Erbrechen von Mageninhalt. |
| | **Kaffeesatz-Erbrechen:** Dunkles, krümeliges Erbrochenes (**Sofortiger Notfall!**). |
| **Darm & Verdauung** | **Bauchkrämpfe:** Plötzlich einschießende, wellenartige Schmerzen im ganzen Bauch. |
| | **Blähbauch:** Der Bauch ist prall, hart und schmerzhaft durch zu viel Luft. |
| | **Durchfall:** Wässriger oder sehr weicher Stuhl (öfter als 3-mal am Tag). |
| | **Verstopfung:** Seltener Stuhlgang (weniger als 3-mal pro Woche) oder sehr harter Stuhl. |
| | **Fettstuhl:** Glänzender, klebriger Stuhl, der in der Toilette oben schwimmt. |
| **Leber & Galle** | **Gelbsucht:** Gelbverfärbung des Augenweiß oder der Haut. |
| | **Gallenkolik:** Heftige, krampfartige Schmerzen im rechten Oberbauch. |
| | **Dunkler Urin:** Urin ist auffällig dunkel (wie Cola oder dunkles Bier). |
| | **Heller Stuhl:** Der Stuhlgang ist sandfarben oder fast weiß entfärbt. |
| **Enddarm** | **Teerstuhl:** Tiefschwarzer, glänzender und klebriger Stuhl (**Inneres Bluten!**). |
| | **Frisches Blut:** Hellrote Blutspuren auf dem Stuhl oder am Toilettenpapier. |
| | **Bleistiftstuhl:** Auffällig dünn geformter Stuhlgang. |
| | **Afterjucken:** Starkes Brennen oder Jucken am Ausgang. |
| **Warnsignale** | **Brettharter Bauch:** Die Bauchdecke ist extrem fest und schmerzhaft bei Berührung (**Lebensgefahr!**). |
| | **Gewichtsverlust:** Deutliche Gewichtsabnahme, ohne dass man eine Diät macht. |
| | **Nachtschweiß:** Man wacht nachts schweißgebadet auf. |
