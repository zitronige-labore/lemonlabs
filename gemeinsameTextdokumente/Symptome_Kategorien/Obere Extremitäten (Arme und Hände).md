# Symptom-Katalog: Obere Extremitäten (Arme und Hände)

| Bereich | Spezifisches Symptom | Detail-Beschreibung für die KI-Logik (Symptom-Checker) |
| :--- | :--- | :--- |
| **Schulter & Manschette** | **Impingement-Syndrom** | Stechender Schmerz beim seitlichen Anheben des Arms (zwischen 60° und 120°). |
| | **Nachtschmerz** | Starkes Pochen in der Schulter; Liegen auf der betroffenen Seite ist unmöglich. |
| | **Knirschen (Krepitation)** | Hörbare oder spürbare Reibegeräusche im Gelenk bei kreisenden Bewegungen. |
| **Oberarm & Muskeln** | **Bizepssehnen-Ruptur** | Plötzlicher "Knall" mit anschließender Vorwölbung des Muskelbauchs ("Popeye-Syndrom"). |
| | **Muskelkater vs. Riss** | Diffuser Druckschmerz nach Sport vs. punktueller, messerscharfer Schmerz bei Bewegung. |
| **Ellbogen (Nerven/Sehnen)** | **Sulcus-Ulnaris-Syndrom** | Elektrisierende Schläge bis in den Ringfinger bei Druck auf die Ellbogen-Innenseite. |
| | **Bursitis (Schleimbeutel)** | Prall-elastische, oft rote und heiße Schwellung direkt auf der Ellbogenspitze. |
| | **Suppinationsschmerz** | Schmerz beim kräftigen Ausdrehen der Hand (z. B. Schraubbewegung). |
| **Unterarm & Faszien** | **Logensyndrom** | Extremer Druckschmerz und Spannungsgefühl nach Verletzungen (**Notfall!**). |
| | **Sehnenscheiden-Reiben** | Tastbares "Schneeballknirschen" entlang der Sehnen am Unterarm bei Bewegung. |
| **Handgelenk & Mechanik** | **TFCC-Läsion** | Schmerz an der Kleinfingerseite des Handgelenks bei Drehbewegungen oder Stützen. |
| | **Karpaltunnel-Atrophie** | Sichtbares "Loch" oder flacher Daumenballen durch langjährigen Nervendruck. |
| | **Fallhand** | Unfähigkeit, die Hand im Handgelenk nach oben zu strecken (Radialis-Lähmung). |
| **Hohlhand & Finger** | **Dupuytren-Kontraktur** | Tastbare Knoten oder Stränge in der Innenhand; Finger lassen sich nicht mehr flach auflegen. |
| | **Trigger-Finger** | Finger bleibt in Beugung hängen und "schnappt" erst bei Kraftaufwand auf. |
| | **Arthrose-Knoten** | Harte, knöcherne Verdickungen an den Fingergelenken (Heberden- & Bouchard-Knoten). |
| **Neurologie & Sensorik** | **Parästhesien** | Brennen, Kribbeln oder "Ameisenlaufen" ohne äußeren Reiz. |
| | **Dysästhesie** | Missempfindung: Normale Berührung wird als unangenehm oder schmerzhaft empfunden. |
| | **Diskriminierung** | Verlust der Fähigkeit, zwei Berührungspunkte getrennt wahrzunehmen. |
| **Gefäße & Trophik** | **Kapillarfüllzeit** | Nach Druck auf den Nagel dauert es länger als 2 Sek., bis er wieder rosa wird. |
| | **Ischämieschmerz** | Unerträglicher Schmerz bei blasser, kalter Haut (**Gefäßverschluss - Notfall!**). |
| | **Schwitzstörung** | Betroffene Hand ist auffällig trockener oder feuchter als die gesunde Seite. |
| **Warnsignale (Rot)** | **Plötzliche Lähmung** | Arm sackt kraftlos ab oder lässt sich gar nicht bewegen (**Schlaganfall-Verdacht**). |
| | **Akute Schwellung** | Einseitig heißer, roter und stark geschwollener Arm (**Thrombose-Verdacht**). |
